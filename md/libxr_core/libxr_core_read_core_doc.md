## 核心组件简介

本章节介绍 LibXR 框架的核心模块，提供跨平台通用的：

- 基础数据类型与类型识别
- 错误码与错误处理约定
- 通用读写接口与操作封装
- 时间与时间差工具
- 终端颜色与 ANSI 控制

这些模块不直接“干业务”，但会在你之后写的所有系统层 / 驱动 / 上层逻辑里频繁出现，是 LibXR 的基础支撑层。

---

## 模块概览：先搞清每个名字大概负责什么

- **libxr_def**  
  常用宏定义、错误码与基础常量。

- **libxr_assert**  
  断言机制与致命错误回调。

- **libxr_cb**  
  类型安全的回调机制。

- **libxr_type**  
  原始数据封装与类型识别。

- **libxr_color**  
  终端输出格式与 ANSI 控制。

- **libxr_time**  
  微秒 / 毫秒级时间戳与时间差。

- **libxr_rw**  
  通用读写接口与操作封装。

> 暂时不要求记 API，只要能一句话概括每个模块“负责哪一块事”。
